- `npm run lint` - Run ESLint

## Architecture

Cinema Roll is a movie recommendation app built with React + Vite, using Supabase for persistence and Claude AI for recommendations.

### External Services

Three APIs are required (configured via environment variables):
- **Supabase** (`VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`) - PostgreSQL database for storing movie lists
- **TMDB** (`VITE_TMDB_API_KEY`) - Movie search and metadata
- **Anthropic** (`VITE_ANTHROPIC_API_KEY`) - AI recommendations via Claude

### Data Flow

1. User searches movies via TMDB API (`src/lib/tmdb.js`)
2. Movies are saved to Supabase with `list_type` of either `to_watch` or `watched` (`src/lib/supabase.js`)
3. Recommendations are generated by sending both lists to Claude API (`src/lib/claude.js`)

### Database Schema

Single `movies` table with `list_type` column differentiating between "to_watch" and "watched" lists. Schema is in `supabase-schema.sql`.

### UI Structure

Three main views controlled by `activeTab` state in `App.jsx`:
- **Recommend** (default) - Hero with AI recommendation button and list stats
- **Lists** - Toggle between To Watch and Watched lists
- **Search** - TMDB movie search to add films

### Styling

Minimalist noir design with copper accent color. Uses DM Serif Display for headings and Inter for body text. All styles in `src/index.css` using CSS custom properties.
